# V0.0.1 Application MarieTeam 
##  PdfSharp 2.4.2
## MySql.Data 8.0.23
## .NET 4.7.2 WPF

* Generateur et editeur de PDF 

## Features :
* Gestion fine des erreurs
* Design style WPF
* Génération de PDF avec images
* Ajouts de bateaux et de pages via l'éditeur txt
* Un bateau par page avec image
* Tests unitaires
